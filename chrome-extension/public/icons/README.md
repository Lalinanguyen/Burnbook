# Icons

To complete the extension setup, you need to add icon files in this directory:

- `icon16.png` - 16x16 pixels (for toolbar/tabs)
- `icon48.png` - 48x48 pixels (for extension management page)
- `icon128.png` - 128x128 pixels (for Chrome Web Store)

For now, you can use any PNG images with these sizes, or create simple icons using an online tool like:
- https://www.favicon-generator.org/
- https://www.canva.com/

Recommended icon design: A book with flames or a similar theme representing "Burn Book"

Colors from the theme:
- Primary: Blue (#3B82F6)
- Severity colors: Green to Red gradient
